STEP 1:

Choose one version of mod 
- CANCER, DEFAULT, DEFAULT_AND_CONSUMABLES, DEFAULT_NO_FAINT, FATIGUE_ONLY, FATIGUE_ONLY_NO_FAINT

Move .pak files to ~mods folder. Path: <your-game-path>\Stalker2\Content\Paks\~mods

If you don't have ~mods folder - CREATE IT.

STEP 2:

IF UE4SS IS NOT INSTALLED:
Move "Binaries" folder to <your-game-path>\Stalker2

IF UE4SS IS INSTALLED:
Move '\Sopor\Binaries\Win64\ue4ss\Mods\SoporCombatSleep' folder to <your-game-path>\Stalker2\Binaries\Win64\ue4ss\Mods
